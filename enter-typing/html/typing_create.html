# 시간(초) 가사 읽기 (히라가나) 작업
등록된 가사가 없습니다. 오른쪽 입력창에서 가사를 추가해주세요.
타이핑 추가 가이드 ✕

1. 영상 불러오기
YouTube 영상의 URL을 입력하고 불러오기 버튼을 눌러 영상을 가져옵니다.

2. 시간 맞추기 (싱크)
영상을 재생하다가 원하는 구절이 시작되는 타이밍에 ⏱️ 싱크 버튼을 누르면 현재 시간이 자동으로 채워집니다.

3. 가사 자동 변환
가사 텍스트를 입력하고 자동 변환 버튼을 누르면 읽기(히라가나)가 알아서 변환됩니다! (직접 수정도 가능합니다)

4. 목록에 추가
내용이 올바른지 확인한 후 + 추가 버튼을 눌러 소절을 등록하세요.

5. 여기서 종료하기 (선택)
노래가 끝난 후 반주가 너무 길다면, 원하는 시간에서 여기서 종료하기 버튼을 눌러주세요. 게임 플레이 시 해당 지점에서 바로 종료되고 점수판으로 이동합니다.

모든 소절 등록을 마친 후 우측 상단의 저장을 누르면 타이핑 등록이 완료됩니다!
